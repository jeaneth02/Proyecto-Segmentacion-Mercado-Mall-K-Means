# 🛍️ Proyecto de Segmentación de Mercado de Mall - K-Means  

## 📋 Descripción del Proyecto  
Proyecto desarrollado en **6 sprints** para identificar segmentos de clientes en un centro comercial mediante análisis de clustering con **K-Means**.  

El objetivo principal es optimizar estrategias de marketing personalizadas, mejorar la experiencia del cliente y aplicar herramientas de gestión de proyectos durante el desarrollo del análisis de datos.

## 🎯 Objetivos del Proyecto  

- Identificar patrones de compra mediante análisis de datos  
- Segmentar clientes usando el algoritmo **K-Means Clustering**  
- Desarrollar estrategias de marketing personalizadas por segmento  
- Aplicar herramientas de gestión de proyectos (EDT, cronogramas y matrices)  
- Implementar análisis de datos y visualización de resultados
