#  Sprint 1 - Planificación e Inicio del Proyecto
**📅 Fechas:** 12 de febrero - 27 de febrero de 2026  
**⏱️ Duración:** 2 semanas (16 días)  
**✅ Estado:** Completado

## 📖 Descripción
Este sprint se centró en la planificación integral del proyecto de segmentación de mercado. Se definieron los entregables iniciales, se estableció la línea base del cronograma, se identificaron riesgos y se configuraron las matrices de gestión para garantizar un seguimiento estructurado durante todo el ciclo de vida del proyecto.

## 💡 Instrucciones de Uso
1. Los archivos `.xlsx` se pueden abrir con **Microsoft Excel**, **Google Sheets** o **LibreOffice**.
2. El archivo `.pdf` contiene la EDT oficial para referencia rápida sin edición.
3. Para visualizar la Curva S o el Valor Ganado, abre los Excel y revisa las pestañas de gráficas/dashboard.
4. Mantén los registros de problemas y lecciones aprendidas actualizados al finalizar cada reunión de seguimiento.
